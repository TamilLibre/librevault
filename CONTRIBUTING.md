# 🤝 Contributing to LibreVault

## 👋 Vanakkam!

LibreVault-ku contribute panna interest kaatradhukku romba nandri! 💚

LibreVault oru **community-driven project**. Namma goal simple — ellarukkum **Free & Open Source Software (FOSS)** discover panna easy-aana oru trusted resource create panradhu.

Perusa contribution pannunga-nu illa. Chinna help kooda project-ku romba valuable. 🚀

---

# 🌟 Eppadi Contribute Pannalam?

Neenga contribute panna sila ways:

- ➕ Pudhu software suggest pannalam
- 🐞 Broken or outdated links report pannalam
- ✏️ Software descriptions improve pannalam
- 🌍 Typos & grammar mistakes fix pannalam
- 📂 Categories organize panna help pannalam
- 📖 Documentation improve pannalam
- 💡 Pudhu ideas & features suggest pannalam

---

# 📋 Contribute Pannradhukku Munnaadi

Software suggest panna munnaadi, mudinja varaikum verify pannunga:

- ✅ Official Website
- ✅ Official Repository (GitHub, GitLab, Codeberg...)
- ✅ License Information
- ✅ Supported Platforms
- ✅ Correct Category

Idhu maintain panna romba easy-a irukkum.

---

# 📝 Software Suggest Pannumbodhu

Mudinja varaikum indha details include pannunga:

- 📛 Software Name
- 📝 Short Description
- 🌐 Official Website
- 🐙 Official Repository
- 📜 License
- 💻 Supported Platforms

Complete information kudutha review process fast-a mudiyum.
