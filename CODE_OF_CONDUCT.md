# ❤️ Code of Conduct

## 👋 Vanakkam!

TamilLibre & LibreVault community-ku welcome! 💚

Indha project-la participate panra ellarum friendly, respectful & helpful environment maintain panna help pannanum-nu nambrom.

Namma goal software mattum build panradhu illa...

Oru nalla community-yum build panradhu. 🚀

---

# 🤝 Namma Expectations

Community members kitta irundhu naanga expect panradhu:

- 💚 Respect-oda interact pannunga
- 💬 Friendly & constructive discussions pannunga
- 🙌 Beginners-ku help pannunga
- 🌱 Knowledge share pannunga
- 🤝 Different opinions-ah respect pannunga
- 📚 Learning mindset maintain pannunga

---

# ❌ Idha Avoid Pannunga

Please avoid:

- 🚫 Personal attacks
- 🚫 Harassment
- 🚫 Hate speech
- 🚫 Discrimination
- 🚫 Spam
- 🚫 Trolling
- 🚫 Misinformation
- 🚫 Illegal content

---

# 🌍 Inclusive Community

TamilLibre ellarukkum open.

Experience level, background, language, operating system, profession... edhuvum mukkiyam illa.

Respect irundha podhum. ❤️

---

# 🚨 Reporting Issues

Yaaravadhu indha Code of Conduct-ah violate pannanga-na,

GitHub Issue create pannunga illa maintainers-ah contact pannunga.

Naanga appropriate action eduppom.
