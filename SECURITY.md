# 🔒 Security Policy

## 👋 Vanakkam!

LibreVault-ku security important.

Software collections, links, documentation, illa repository-la yedhavadhu security-related issue kandupidicha, public-ah post pannama maintainers-ku report pannunga.

---

# 🚨 Report a Security Issue

Security issue report panna:

- GitHub Issue create pannama irunga.
- Maintainers-ah direct-ah contact pannunga.
- Issue pathi clear details share pannunga.
- Mudinja varaikum reproduce panna steps include pannunga.

---

# 📋 Report-la Include Pannunga

Please include:

- 📝 Issue description
- 📍 Affected page/file
- 🔄 Steps to reproduce (if applicable)
- 📷 Screenshots (if applicable)
- 💡 Suggested fix (optional)

---

# ⚡ Response Time

Report receive pannadhum:

- ✅ Acknowledge panna try pannuvom.
- 🔍 Issue verify pannuvom.
- 🛠️ Fix prepare pannuvom.
- 📢 Update kuduppom.

Response time issue severity-a poruthu vary aagalam.

---

# 💚 Responsible Disclosure

Security issues public-ah share pannradhukku munnaadi maintainers-ku report pannunga.

Idhu project-ahum users-ahum safe-ah maintain panna help pannum.

LibreVault secure-a irukka help pannradhukku nandri! 💚
